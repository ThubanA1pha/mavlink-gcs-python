# 데모 시나리오 가이드

## 1. 목적

이 문서는 MAVLink GCS 프로토타입을 시연하기 위한 기본 데모 절차를 정리한 문서입니다.

데모의 핵심은 다음 기능을 보여주는 것입니다.

- MAVLink 기반 SITL 연결
- 실시간 텔레메트리 표시
- 지도 기반 목표 이동
- 비행금지구역 설정
- A* 기반 회피 경로 생성
- 웨이포인트 미션 계획
- 안전도 모니터링
- 긴급 대응 버튼
