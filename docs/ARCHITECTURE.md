# 시스템 구조 문서

## 1. 개요

본 프로젝트는 Python 기반 MAVLink GCS 프로토타입입니다.

주요 목적은 드론의 텔레메트리 수신, 지도 기반 임무 계획, 비행금지구역 회피 경로계획, 실시간 안전도 모니터링을 하나의 GCS 애플리케이션에서 통합적으로 처리하는 것입니다.

---
