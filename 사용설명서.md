# MAVLink GCS 사용자 설명서

## 1. 프로그램 개요

MAVLink GCS는 Python 기반의 드론 지상관제소(Ground Control Station) 프로토타입입니다.

이 프로그램은 ArduPilot SITL 또는 Mission Planner SITL 환경과 MAVLink로 연결하여 드론의 상태를 실시간으로 확인하고, 지도 기반 미션 계획, 웨이포인트 관리, 비행금지구역 회피, 임무 안전도 모니터링 기능을 제공합니다.

본 프로그램은 학습, 연구, 시뮬레이션 검증 목적의 소프트웨어이며 실제 기체 운용 전에는 반드시 충분한 검증이 필요합니다.

---

## 2. 실행 전 준비 사항

### 2.1 필요 환경

- Windows 10 또는 Windows 11
- Python 3.10 이상
- ArduPilot SITL 또는 Mission Planner SITL
- MAVLink TCP 연결
- Python 패키지 설치 환경

### 2.2 Python 패키지 설치

프로젝트 폴더에서 다음 명령을 실행합니다.

```bash
pip install -r requirements.txt
