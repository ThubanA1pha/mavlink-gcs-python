# GCS 드론 관제 프로그램 사용설명서

> 대상 파일: `GCS_STEP8.py` (비행금지구역 회피 경로계획 포함 최신본)
> ArduCopter SITL 시뮬레이터 + 지도 기반 지상관제소(GCS)

---

## 1. 프로그램 개요

지도 위에서 SITL(소프트웨어 시뮬레이션 드론)을 띄우고, 실시간 상태(HUD)를 보며 마우스·조이스틱·버튼으로 드론을 제어하는 프로그램이다. 핵심 기능:

- 지도 표시 + 드론/홈 마커 + HUD(자세·배터리·고도·속도·모드)
- 비행 모드 변경 / 시동(ARM)·해제(DISARM) / 이륙(TAKEOFF)
- 마우스 클릭으로 목표 지점 이동(GUIDED)
- 가상 조이스틱으로 직접 조종(RC)
- **비행금지구역(다각형)을 그려두면, 목표 이동 시 구역을 회피하는 경로를 자동 생성해 순차 비행**

## 3. 화면 구성

```
┌─────────────────────────────────────────────┬───────────────┐
│  [Zoom:18]      LAT[ ] LON[ ] [Start SITL]   │ 비행 모드 설정 │
│                                  ┌────HUD───┐ │  STABILIZE … │
│                                  │ 인공수평선 │ │ 기체 제어/상태│
│         지   도   (좌측)          │ 배터리/모드│ │  ARM/DISARM  │
│            드론·홈·목표 마커        └──────────┘ │  TAKEOFF     │
│                                              │ 기타(배터리)  │
│            [가상 조이스틱]                     │ 비행금지구역  │
│                                              │ 지도 중심     │
│  상태바: 실시간 데이터 수신 중 …               │ 가상 조이스틱 │
└─────────────────────────────────────────────┴───────────────┘
```

- **상단**: 위도(LAT)/경도(LON) 입력칸 + **Start/Stop SITL 토글 버튼**(시작=노랑, 중지=빨강).
- **좌측 지도**: 좌상단에 줌 레벨, 우상단에 HUD, 하단 중앙에 조이스틱(켤 때만).
- **우측 제어 패널**: 모드/제어/금지구역/지도중심/조이스틱.
- **하단 상태바**: 현재 진행 상황·위치 표시.

### HUD 읽는 법

- **인공 수평선**: 파랑=하늘, 갈색=지면. 기울면 롤(Roll), 위아래로 움직이면 피치(Pitch).
- **상태**: `ARM`(초록)=시동 걸림 / `DISARM`(빨강)=꺼짐.
- **배터리**: `전압V (잔량%)`, **위성수**, **지상속도(m/s)**, **상대고도(m, AGL)**, **모드**.

## 4. 기본 사용 순서 (필수 흐름)

1. 프로그램 실행 → **지도 선택 팝업**에서 지도 종류 고르고 **[확인 및 실행]**.
2. 상단 LAT/LON에 시작 위치 입력(기본값=서울시청). → **[Start SITL]** 클릭.
3. **약 20초 부팅** 대기 → 연결되면 HOME 마커 생성.
4. **GPS 초기화 10초** 카운트다운 팝업 대기 → "준비 완료" 뜨면 드론 마커 등장.
5. 우측 패널에서 비행 모드 → 시동 → 이륙 → 이동 순으로 조작.
6. 끝나면 **[Stop SITL]**(다시 시작 가능) 또는 창 닫기로 종료.

> **중요 순서**: 모드 설정 → ARM(시동) → TAKEOFF(이륙) → 이동. 이륙 전(지상)에는 이동 명령이 거부된다.

## 5. 기능별 사용법

### 5.1 비행 모드 설정

- 버튼: `STABILIZE` `ALT_HOLD` `LOITER` `GUIDED` `AUTO` `RTL` `LAND`.
- 클릭하면 해당 모드로 전환 명령 전송. 전환되면 상태바/HUD의 모드 표시가 바뀐다.
- **시동(ARM)은 STABILIZE / LOITER / GUIDED / ALT_HOLD 에서만** 가능(나머지 모드에서는 경고).

### 5.2 시동·이륙

- **[ARM (시동)]**: 모터 시동. 허용 모드가 아니면 안내창이 뜬다.
- **[DISARM (해제)]**: 시동 해제(착륙 상태에서).
- **[TAKEOFF (이륙)]**: 고도 입력창 → 입력한 고도로 이륙(필요 시 GUIDED로 자동 전환).
- 우측 **상태 라벨** 색으로 현재 가능 여부 안내: 연결대기 / 시동 불가(모드) / PreArm 점검중 / READY(시동 가능) / ARMED(비행 중).

### 5.3 마우스 클릭으로 이동 (GUIDED)

1. **이륙한 상태**(ARM + 고도 ≥ 0.5m)에서 지도의 목표 지점을 **왼쪽 클릭**.
2. 목표(TARGET) 마커 생성 → **이동 고도 입력창**.
3. **금지구역이 없으면**: 그 지점으로 직선 GUIDED 이동(빨간 선 표시).
4. **금지구역이 있으면**: 구역을 회피하는 경로를 자동 계산해 **초록 경로선**으로 그리고 웨이포인트를 순차 비행.

### 5.4 비행금지구역 그리기 + 회피 (핵심 기능)

1. 우측 **[금지구역 그리기]** 버튼을 누르면 **그리기 모드 ON**(버튼이 빨강 "그리기 ON"으로 바뀜).
2. 지도를 클릭할 때마다 **정점**이 찍힌다(번호 마커). **3개 이상** 찍으면 미리보기 폴리곤(주황)이 보인다.
3. **[구역 확정]** → 빨간 금지구역으로 확정(여러 개 만들 수 있음).
4. **[금지구역 그리기]**를 다시 눌러 **그리기 모드 OFF**.
5. 이륙 후 구역 **건너편**을 클릭 → 고도 입력 → **경로가 구역을 우회**해서 그려지고 순차 비행.
6. **[전체 초기화]**: 모든 금지구역·경로를 삭제.

> 경로 계획이 실패하면(예: 목표가 구역 안, 이륙 전) 화면 중앙에 토스트 안내가 뜬다.

### 5.5 지도 중심 자동 이동

- **이동 없음**: 지도가 고정.
- **홈 중심**: LAT/LON 위치로 지도 1회 이동.
- **드론 추적**: 1초마다 지도를 드론 위치로 따라 이동.

### 5.6 가상 조이스틱 (직접 조종)

1. 조건: **ARM(시동) + LOITER 또는 ALT_HOLD 모드**. (아니면 안내 후 자동 해제)
2. 우측 **[조이스틱 표시 / 조종]** 체크 → 지도 하단에 두 개의 스틱 표시.
3. **왼쪽 스틱**: 위/아래=스로틀(상승/하강), 좌/우=요(좌/우 회전).
4. **오른쪽 스틱**: 위/아래=피치(전진/후진), 좌/우=롤(좌/우 이동).
5. 마우스를 떼면 스틱은 중립으로 복귀(제자리 정지).
6. 비행 조건이 깨지면(시동 해제·모드 변경) 조이스틱이 **자동 차단**된다.

### 5.7 기타

- **[배터리 초기화]**: 시뮬레이터 배터리 잔량을 100%로 리셋(시동 중이면 경고 확인).
- **착륙 자동 정리**: 시동이 해제되면 약 2초 뒤 목표 마커·경로선이 자동으로 지워진다.

## 6. 종료

- **[Stop SITL]**: 시뮬레이터만 끄고 창은 유지 → 다시 Start 가능.
- **창 닫기(X)**: 프로그램 완전 종료.
- 권장: Stop SITL로 정리 후 창 닫기.

## 7. 문제 해결 (FAQ)

| 증상 | 원인 / 해결 |
|---|---|
| Start SITL 시 `FileNotFoundError [WinError 2]` | `sitl/ArduCopter.exe`가 없음. `sitl` 폴더를 프로그램과 같은 폴더에 둘 것. |
| 콘솔에 `[MAVLINK ERROR] [WinError 10053]` | 대부분 **종료/Stop 시 정상적인 잡음**. 운영 중 계속 뜨면 SITL이 꺼졌는지 확인. |
| 지도가 회색/안 보임 | 인터넷 연결 확인. 한 번 본 영역은 `map/` DB에 저장돼 다음엔 빠름. |
| 드론 마커가 안 뜸 | Start 후 **부팅 20초 + GPS 초기화 10초**를 기다릴 것. |
| ARM이 안 됨 | 모드가 STABILIZE/LOITER/GUIDED/ALT_HOLD인지, 상태 라벨이 READY인지 확인. |
| 클릭해도 이동 안 함 | **이륙 후**(고도 ≥ 0.5m)에만 이동 가능. 시동·이륙 먼저. |
| 조이스틱 체크가 자동 해제됨 | ARM + LOITER/ALT_HOLD 조건이 아님. 모드 변경 후 다시 시도. |
| 경로 계획 실패 토스트 | 목표가 금지구역 내부이거나, 안전여유 안쪽, 또는 이륙 전. 목표 위치/상태 확인. |
| 금지구역이 너무 빨갛게 지도를 가림 | 의도된 표시. 외곽선만 원하면 코드의 `finalize_zone`에서 `fill_color`를 조정. |

## 8. 데모 추천 시나리오 (비행금지구역 회피)

1. Start SITL → 부팅·GPS 대기.
2. **금지구역 그리기 ON** → 지도에 사각형(정점 4개) → **구역 확정** → 그리기 OFF.
3. **LOITER** 모드 → **ARM** → **TAKEOFF**(예: 고도 10m).
4. 구역 **반대편**을 클릭 → 고도 입력 → 드론이 **구역을 우회**해 순차 비행하는지 확인.
5. 비교: 금지구역을 지우고(전체 초기화) 같은 지점을 클릭하면 **직선으로 관통**(대조).
